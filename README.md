# Nuxt 4 Starter

## Overview

Nuxt 4 Starter Auth is a template for a Nuxt 4 application.

## Features

- Nuxt4 Blank Template
- ESLint with antfu config
- Prettier
- Husky
- Lint-staged
- Tailwind CSS (Coming soon)

## Tech Stack

- Frontend: Nuxt 4
- Database: PostgreSQL
- UI Framework: Tailwind CSS
